# User Persona

## Persona 1: 品牌运营实习生

背景：

- 参与新品上市、内容 Brief、竞品监测和消费者洞察整理
- 需要经常把评论、问卷和社媒反馈写进周报

痛点：

- 评论太多，人工整理耗时
- 不确定哪些反馈值得被写成 insight
- 需要把用户原话转化成品牌能用的策略语言

需要的功能：

- 情绪识别
- 高频痛点
- 卖点提炼
- 周报式洞察总结

## Persona 2: 电商运营

背景：

- 负责天猫/京东商品标题、详情页、活动页和数据复盘

痛点：

- 用户评论和客服反馈没有被系统利用
- 价格、赠品、物流和详情页表达容易造成用户困惑
- 大促后需要快速沉淀复盘结论

需要的功能：

- 负反馈识别
- 商品表达优化建议
- 活动规则反馈分析
- 详情页卖点优化建议

## Persona 3: 内容运营

背景：

- 负责小红书、抖音、公众号选题和内容发布

痛点：

- 选题容易凭感觉
- 爆款内容和真实用户痛点连接不够
- 缺少从评论到内容选题的稳定方法

需要的功能：

- 用户问题归类
- 内容选题生成
- 平台化表达建议
- 代表性用户原话
