# AI Workflow

## 工作流名称

消费者评论到运营洞察工作流

## 流程

1. 评论导入
   - 来源：天猫、京东、小红书、抖音、问卷开放题
   - 格式：商品名、评分、评论文本

2. 数据清洗
   - 删除空评论
   - 保留用户原话
   - 避免使用真实隐私数据

3. 情绪识别
   - 正面：满意、推荐、回购、认可
   - 中性：描述事实、评价模糊
   - 负面：抱怨、疑虑、体验问题

4. 主题归类
   - 成分透明度
   - 价格/性价比
   - 产品效果
   - 包装设计
   - 物流配送
   - 客服体验

5. 痛点提取
   - 从负面评论提炼用户核心诉求
   - 保留代表性原话，方便人工复核

6. 洞察生成
   - 输出整体情绪
   - 输出高频主题
   - 输出典型痛点
   - 输出运营建议

7. 内容转化
   - 小红书：用户问题 + 场景化表达
   - 抖音：短视频脚本和快速解释
   - 公众号：深度复盘和购买决策指南
   - 直播间：FAQ 和购买顾虑处理

8. 人工复核
   - 标记可采纳、需修改、不可用
   - 记录人工修正原因
   - 反向优化 Prompt 和分类规则
